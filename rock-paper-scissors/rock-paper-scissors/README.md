# Rock Paper Scissors

Classic Rock-Paper-Scissors game with beautiful ASCII art and clean code.

### How to Play
- Type **0** for Rock
- Type **1** for Paper  
- Type **2** for Scissors

Rock beats Scissors  
Scissors beats Paper  
Paper beats Rock

### Features
- Full ASCII art for all three choices
- Input validation (no crashes if you type wrong)
- Clear visual result with separators
- Shows both choices at the end
- Professional look – perfect for junior Python portfolios 2025–2026

### How to Run
```bash
python main.py
