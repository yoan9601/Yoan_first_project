# Treasure Island – Text Adventure Game

Classic "choose your own adventure" game with multiple endings.  
Only one path leads to the treasure!
left → wait → yellow = YOU WIN!
### Features
- Fully interactive story with 3 critical decisions
- Beautiful ASCII art title screen
- 7 different endings (1 win + 6 dramatic Game Over scenarios)
- Clean, well-commented Python code
- Zero external dependencies

### Winning Path
1. At the crossroad → **left**  
2. At the lake → **wait** for the boat  
3. At the house → choose the **yellow** door

python main.py
